Projet de session
Solo

Description: Jeu 2D de tir/survie fais sur Unity et quelque assets(Sounds/Sprites/Animations) trouver sur Itch.io

Pour jouer au jeu 2D de tir/survie, veuillez suivre les étapes suivantes :

Téléchargez les fichiers de jeu en format .zip
Extrayez le contenu du fichier .zip dans un dossier sur votre ordinateur.
Ouvrez le fichier FireDude.exe pour lancer le jeu.
Utilisez les touches fléchées ou les touches WASD pour contrôler le joueur, et la souris pour viser et tirer.
Vous pouvez trouver les scripts utilisés pour développer le jeu dans le dossier "Assets/Scripts".
